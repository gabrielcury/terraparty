This guide assumes you have the AWS CLI installed and credentials on your system already.
We are using Terraform with JSON syntax in this guide. To switch your output to HCL (Hashicorp Configuration Language) check out this link

  1. Install Terraform if you haven't already
  2. Throw the exported json file into a new folder
  3. Run terraform init from the command line in that folder
  4. Run terraform plan to tell you what Terraform will do
  5. Run terraform apply to actually deploy these resources to the cloud. You will be charged for this!
  6. Run terraform destroy to delete all those resources. It doesn't hurt to double check in your cloud provider account and other regions too.
