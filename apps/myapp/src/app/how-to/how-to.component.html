This guide assumes you have the AWS CLI installed and credentials on your system already.

  1. Create a new folder
  2. Add exported json file to that folder
  3. Install Terraform if you haven't already
  4. Run terraform init from the command line in that folder
  5. Run terraform plan to tell you what Terraform will do
  6. Run terraform apply to actually deploy these resources to the cloud. You will be charged for this!
  7. Run terraform destroy to delete all those resources. Doesn't hurt to double check in your cloud provider account and other regions too.
